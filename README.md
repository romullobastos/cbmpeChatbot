# CBMPE Digital

Sistema de atendimento digital do Corpo de Bombeiros Militar de Pernambuco.

## Sobre o Projeto

O CBMPE Digital é uma plataforma web responsiva que permite aos cidadãos solicitar serviços do Corpo de Bombeiros de forma rápida e digital, sem necessidade de deslocamento.

## Funcionalidades

- **Assistente Virtual (Chatbot)** - Atendimento automatizado 24h para solicitação de serviços
- **Emissão de Certificados** - AVCB, CLCB, Projetos Técnicos e Vistorias
- **Agendamento de Vistorias** - Escolha data e horário para vistoria técnica
- **Consulta de Protocolo** - Acompanhe o status da sua solicitação
- **Contato com Atendente** - Solicite contato humano quando necessário

## Serviços Disponíveis

| Serviço | Descrição |
|---------|-----------|
| AVCB | Auto de Vistoria do Corpo de Bombeiros |
| CLCB | Certificado de Licenciamento do Corpo de Bombeiros |
| Projetos | Análise de Projetos Técnicos de Segurança Contra Incêndio |
| Vistorias | Agendamento de Vistoria Técnica |

## Tecnologias

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Radix UI (componentes acessíveis)
- Lucide React (ícones)

## Como Executar

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Build para produção
npm run build
```

## Layout

O sistema é totalmente responsivo:

- **Mobile**: Interface otimizada para smartphones com navegação simplificada
- **Desktop**: Layout com sidebar lateral e área de conteúdo expandida
